# AMRAdapterPangle

Pangle adapter for AMRSDK. Allows publishers to mediate Pangle banner, native, interstitial and video ads in AMRSDK.

## Versioning

Versions follow `MAJOR.MINOR.PATCH` where `MAJOR.MINOR.PATCH` tracks the Pangle (Ads-Global) SDK version.

## Requirements

- iOS 12.0+
- Xcode 15+

## Installation

### Swift Package Manager

In Xcode: **File → Add Package Dependencies** and enter the repository URL:

```
https://github.com/admost/AMR-IOS-ADAPTER-PANGLE.git
```

### CocoaPods

Add the following to your `Podfile`:

```ruby
pod 'AMRAdapterPangle'
```

Then run:

```sh
pod install
```
